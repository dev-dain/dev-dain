- 필요한 기술을 배우고, 빠르게 구현한 뒤, 꼼꼼히 수정하고 유지하면서 앱이 살아있도록 만듭니다.
- 클라이언트와 가장 가까이 있는 개발자로서, 클라이언트와 소통하고 훌륭한 사용자 경험을 제공하기 위해 노력합니다.
